# Rules for using the context files in this directory

1. these files are general guidelines for writing a screenplay, and shouldn't have direct correlation to the source material
2. refer to the [premise](../01-premise.md) for the overall story and tone
3. refer to the [outline](../06-Outline-limited-series.mdx) for the outline

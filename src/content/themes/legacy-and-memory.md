# Legacy and Memory

## Core Concept

This theme examines how institutions protect their legacy by controlling historical narrative, creating an official version of events that may bear little relation to reality. It explores the gap between personal memory and institutional history, and shows how the final act of institutional power is not the exercise of force, but the erasure of inconvenient truths.

## Institutional Memory Control

### Official Narratives
- **Title Card Denial**: Final statement denying royal involvement in Lennon's death
- **Sealed Files**: MI5 documents classified to protect institutional reputation
- **Official Histories**: Records that preserve institutional image over factual accuracy

### Personal vs. Institutional Memory
- **Nigel and Clive**: Personal memory of events vs. institutional erasure
- **Philip's Humiliation**: Personal memory of failure vs. institutional continuity
- **Queen's Wisdom**: Understanding that legacy matters more than current events

### Historical Revisionism
- **"No Evidence"**: Institutional strategy of denying what can't be proven
- **Plausible Deniability**: Creating conditions where truth can never be established
- **Career Management**: Moving personnel to prevent institutional memory from forming
